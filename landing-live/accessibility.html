Erişilebilirlik Beyanı

Kavira Software platformu, engelli kullanıcılar dahil herkes tarafından erişilebilir olmayı amaçlar.

1. Bağlılığımız

Platform, WCAG 2.1 Düzey AA standartlarına uyum hedefiyle geliştirilmektedir. Görme, işitme, motor ve bilişsel engellilere yönelik temel erişilebilirlik kriterlerini sağlamayı taahhüt ederiz.

2. Uygulanan Önlemler

  • Klavye ile tam gezilebilirlik (Tab, Enter, Esc)
  • Anlamsal HTML yapısı (heading hiyerarşisi, landmark roles)
  • Form alanlarında label ve aria-* etiketleri
  • WCAG önerilen renk kontrast oranları (en az 4.5:1)
  • Ekran okuyucu uyumlu metin yapısı
  • Responsive tasarım (mobil, tablet, masaüstü)
  • 200% zoom'a kadar kayıpsız okunabilirlik
  • prefers-reduced-motion desteği — animasyon azaltma tercihine uyar

3. Bilinen Sınırlamalar

  • Flutter Web tabanlı bazı arayüzlerde ekran okuyucu desteği sınırlıdır; iyileştirme devam ediyor.
  • Hero canvas animasyonu sembolik amaçlıdır; metinsel alternatif sunulmaktadır.

4. Test Ortamı

Platform aşağıdaki teknolojilerle test edilmektedir:

  • NVDA, JAWS (Windows ekran okuyucu)
  • VoiceOver (macOS / iOS)
  • TalkBack (Android)
  • Chrome, Firefox, Edge, Safari güncel sürümleri
